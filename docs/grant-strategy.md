# Grant Strategy

Polymarket UI Kit is a grant candidate because it can expand the ecosystem
surface area:

- More teams can embed markets in media and research products.
- Builder-code-aware components make attribution visible.
- Share cards create reusable launch and distribution assets.
- Public hooks lower the time from idea to prototype.

## Evidence to collect

- GitHub stars.
- npm downloads.
- registry installs.
- demo page visits.
- X impressions and reposts.
- screenshots from apps using the kit.
- issues or PRs from external builders.

## Submission angle

"Polymarket has strong APIs. This project gives builders the frontend primitives
to distribute prediction markets into new products."
