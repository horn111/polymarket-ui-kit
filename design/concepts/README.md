# Polymarket UI Kit — premium cinematic concepts

This package is the approval gate for the premium redesign. It does not change the
production frontend. Every direction uses the same product copy, market fixture, and
surface inventory so the review is about art direction rather than content.

## Shared product frame

- Master brand: **Polymarket UI Kit**
- Edition: **Civic Forecast / Politics edition**
- Hero: **Make probability feel credible.**
- Primary action: **Try Studio**
- Market fixture: 2028 U.S. presidential election, clearly marked as sample context
- Required surfaces on every board: desktop hero, 390 px mobile, MarketCard, Studio,
  mini-identity, palette, material rules, and a six-beat web-motion storyboard
- Theme strategy: dark-primary with a complete light counterpart in the final system
- Politics remains neutral: no party-coded red/blue pair

## Direction A — Signal Optics

**Idea:** probability behaves like an optical instrument. Evidence changes the way a
signal refracts; the interface makes uncertainty visible without becoming decorative.

- Voice: precise, atmospheric, lucid
- Palette: obsidian, smoked cyan, ice light, mint live, coral negative
- Type candidates: Geologica Variable + Azeret Mono (OFL candidates; verify and host
  locally only after approval)
- Mark: four calibrated aperture blades around a quiet center
- Material: smoked glass is limited to hero/feature moments; product surfaces stay
  opaque and readable
- Motion: long optical focus, restrained parallax, line refraction, 180–900 ms beats
- 3D motif: a probability aperture whose opening follows a 0..1 value

Best when the desired impression is a scientific instrument with cinematic lighting.

## Direction B — Broadcast Intelligence

**Idea:** the product feels like a premium live information network. Directional type,
signal rails, and editorial timing create urgency without news-channel clutter.

- Voice: immediate, informed, directional
- Palette: carbon, warm signal white, chartreuse, mint live, orange-red negative
- Type candidates: Barlow Condensed Variable + Public Sans (OFL candidates; verify and
  host locally only after approval)
- Mark: two opposing signal brackets resolving into a probability tick
- Material: matte broadcast panels, hard cuts, luminous scan fields, no glass cards
- Motion: wipes, data locks, live-ticker acceleration, 120–720 ms beats
- 3D motif: a folded signal ribbon carrying chart history through depth

Best when the desired impression is launch-film energy and a live data broadcast.

## Direction C — Mechanical Probability

**Idea:** a market is treated as a physical analytical machine. Outcomes sit on
engraved plates; probability moves with the weight and precision of a shutter.

- Voice: engineered, tactile, assured
- Palette: graphite, anodized metal, ceramic ink, burnished copper, mint live
- Type candidates: Unbounded Variable + Chivo Mono (OFL candidates; verify and host
  locally only after approval)
- Mark: interlocking P/UI plates forming a calibrated gate
- Material: anodized metal, ceramic labels, fine engraving, controlled specular light
- Motion: shutter travel, weighted snaps, plate alignment, 160–840 ms beats
- 3D motif: two machined probability plates that open and rebalance around the current
  market value

**Approved direction:** use Mechanical Probability as the production foundation, with
Broadcast Intelligence's data-lock timing and Signal Optics' restrained chart light.
Purple/violet and chartreuse are explicitly excluded from the master palette.

Best when the desired impression is a collectible instrument rather than software.

## Comparison frame

| Criterion | Signal Optics | Broadcast Intelligence | Mechanical Probability |
| --- | --- | --- | --- |
| First impression | Scientific cinema | Live information network | Tactile precision object |
| Strongest motion asset | Refraction/aperture | Wipes/ribbons/data locks | Shutter/plate mechanics |
| Product UI risk | Too ethereal | Too busy | Too stylized |
| Guardrail | Opaque data surfaces | Strict density limits | Flat fallback for dense UI |
| Future Crypto/Sports fit | High | High | High |
| Portfolio distinctiveness | High | Very high | Very high |

## Approval rule

Select one direction as the system. At most two named techniques may be borrowed from
the other boards. After approval, the selected direction is translated into
`PRODUCT.md`, `DESIGN.md`, production tokens, React components, demo, Studio, docs,
Storybook, registry, examples, and OG assets.
